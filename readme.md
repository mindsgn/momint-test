# momint test

## Getting Started


### 1. clone repository
```
    git clone https://github.com/mindsgn/momint-test

    cd momint-test
```

### 2. change into the directry
```
    cd momint-test
``` 

### 3. set up .env
```
    touch .env
```

### 4. add the following into .env
```
    PORT=5000
    MONGO_URL=mongodb://127.0.0.1:27017/momint
```

### 5. install dependencies
```
    yarn
```

### 6. run application
```
    yarn  start
```
